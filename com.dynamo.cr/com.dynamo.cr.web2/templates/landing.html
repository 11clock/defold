Create Games Like A Pro

No Setup

The Defold Editor is the only software you need to install to create your games. It runs on Windows, OS X and Linux.

Flexible

The engine builds on simple concepts used to create complex things. It is extremely data-driven and flexible. The game logic and rendering pipeline are scripted in Lua.

Fast Iterations

The time for a change to appear in the running game is less than a second, even on your iOS device. Prototype, build, polish and iterate.

Collaboration

From the editor, you can work with your friends through our servers. Add them to your project in the Dashboard and you are all set. Your team doesn't even need to have the same OS.

iOS Development

Through the Defold App, your iOS device connects to the editor and you can use it to test and tweak your game in real-time.

Tool Set

We have specialized tools for creating 2D games. Features include tiled graphics and animated sprites. Collision shapes can be automatically created from image data.

Support

We work hard to get you up to speed as quickly and painless as possible. We do this by providing competent support in the forums as well as frequently updating and extending the documentation.

Progressive

Defold is run by passionate game developers. We are making a service we would love to use ourselves. We have a long list of exciting ideas and features to come in the future.
