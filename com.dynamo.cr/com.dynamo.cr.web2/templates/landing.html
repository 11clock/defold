Create Games Like A Pro

No Setup

The Defold Editor is the only software you need to install to create your games. It runs on Windows, OS X and Linux.

Flexible

The building blocks are simple, but can be used to create complex features. The game logic and rendering pipeline are scripted in Lua.

Fast Iterations

Edit scripts and other content and see the changes take effect in a second on your iPad or iPhone. Prototype, build, polish and iterate.

Collaboration

You don't need any extra software or service to work with your friends. Content can be synchronized in the editor.

iOS Development

Install the Defold App on your iOS device and it connects to the editor. Then you can work on the game in real-time on your device.

Tool Set

We have specialized tools for creating 2D games. Features include tiled graphics and animated sprites. Collision shapes can be automatically created from image data.

Support

We have a lot of experience in game development and are happy to share it with you in the forums.

Progressive

Defold is run by passionate game developers. We are making a service we would love to use ourselves. We have a long list of exciting ideas and features to come in the future.
